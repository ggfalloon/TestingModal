# TestingModal

## Description
An exercise intended for A/B testing using a modal pop up to deteremine click activity on the call to action button from the user.

## Technologies
* JavaScript
* jQuery
* Google Fonts (Lato)
* Ouibounce
* FlipClock
* AJAX
